Div stands “division” or “section”. It is pretty much “made up” tag that does not do anything visible, it just divides document into sections so it is convenient for you to work with document by splitting it in different sections. For example you can have a page with image and text under the image.

In the HTML code, we use the button tag to create the button. In the CSS code, we define a class . my-button to apply the styling rules.

Buttons could be created using different hover.

To make a button hover, you can use CSS. You can add a hover effect to a button by setting the button's CSS :hover pseudo-class. This CSS code will cause the button to change its background color and text color when the user hovers their mouse over it.
